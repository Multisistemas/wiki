====== 1. OBJETIVO ======

Definir la metodología para identificar, determinar, clasificar y definir a las partes interesadas que son pertinentes al Sistema de Gestión de Calidad.

====== 2. ALCANCE ======

Aplica a todos los procesos del Sistema de Gestión de Calidad.

====== 3. DEFINICIONES ======

==== 3.1. Organización: ====

Persona o grupo de personas que tienen sus propias funciones con responsabilidad, autoridades y relaciones para lograr sus objetivos.

==== 3.2. Parte Interesada: ====

Persona u organización que puede afectar, verse afectada o percibirse como afectada por una decisión o actividad.

//NOTA: Las partes interesadas pueden ser internas o externas a la organización.//
==== 3.3. Cliente: ====

Persona u organización que podría recibir o que recibe un producto o un servicio destinado a esa persona u organización o requerido por ella.

//NOTA: Un cliente puede ser interno o externo a la organización.//
==== 3.4. Proveedor: ====

Persona u organización que proporciona un producto o un servicio.

//NOTA: Un proveedor puede ser interno o externo a la organización.//
==== 3.5. Requisito: ====

Necesidad o expectativa establecida, generalmente implícita u obligatoria.

//NOTA 1: “Generalmente implícita”, significa que es habitual o prácticamente común para la organización y las partes interesadas, el que la necesidad o expectativa bajo consideración está implícita.//

//NOTA 2: Los requisitos pueden ser generados por las diferentes partes interesadas o por la propia organización.//

//NOTA 3: Para lograr una alta satisfacción del cliente, puede ser necesario cumplir una expectativa de un cliente, incluso si no está declarada ni generalmente implícita, ni es obligatoria.//
==== 3.6. Requisito Legal: ====

Requisito obligatorio especificado por un organismo legislativo.

==== 3.7. Requisito Reglamentario: ====

Requisito obligatorio especificado por una autoridad que recibe el mandato de un órgano legislativo.

==== 3.8. Sistema de Gestión de Calidad: ====

Conjunto de elementos de una organización interrelacionados o que interactúan para establecer políticas, objetivos y procesos para lograr estos objetivos.

====== 7. ANEXOS ======

===== ANEXO 1. Identificación, y Determinación de los Requisitos de las Partes Interesadas. =====

=== 1. Identificación de las Partes Interesadas. ===

Las partes interesadas son cualquier individuo, grupo u organización que forme parte o se vea afectado por el mismo, obteniendo algún beneficio o perjuicio.

Cada una de las partes interesadas pueden influir en la organización, ya sea de forma directa o indirecta. Para la identificación de estas partes interesadas, se utiliza la caracterización de cada proceso de Multisistemas e Inversiones.

=== 2. Determinación de los Requisitos de las Partes Interesadas. ===

Para una buena gestión de las partes interesadas resulta fundamental entender y determinar los requisitos pertinentes de las partes interesadas, estos pueden ser:

  * Legales
  * Reglamentarios

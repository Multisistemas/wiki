====== 1. OBJETIVO ======

Asegurar que las salidas y los servicios que no cumplen los requisitos de calidad (no conformes) se identifiquen y controlen para evitar su entrega no intencionada al cliente. Los requisitos de calidad son especificados por nuestros clientes, por regulaciones legales, standares establecidos dentro de Multisistemas o por normas técnicas y otros que son determinados por Multisistemas.

====== 2. ALCANCE ======

<font 11.0pt/11;;inherit;;inherit>Cubre desde la detección de salidas y servicios no conformes hasta el establecimiento de las medidas pertinentes que prevengan la insatisfacción del cliente.</font>

<font 11.0pt/11;;inherit;;inherit>Incluye la detección de las salidas de los procesos de prestación de los servicios a los servicios entregados no conforme dentro y fuera de Multisistemas.</font>

====== 3. DEFINICIONES ======

==== 3.1 Servicio: ====

<font 11.0pt/11;;inherit;;inherit>Salida de una organización con al menos una actividad, necesariamente llevada a cabo entre la organización y el cliente.</font>

**<font 11.0pt/11;;inherit;;inherit>Nota 1:</font>**<font 11.0pt/11;;inherit;;inherit>los elementos dominantes de un servicio son generalmente intangibles.</font>

**<font 11.0pt/11;;inherit;;inherit>Nota 2</font>**<font 11.0pt/11;;inherit;;inherit>: los servicios con frecuencia involucran actividades en le interfaz con el cliente para establecer requisitos del cliente así como durante la entrega del servicio, y puede involucrar una relación continua, por ejemplo con bancos, entidades contables u organizaciones públicas, como escuelas u hospitales públicos.</font>

**<font 11.0pt/11;;inherit;;inherit>Nota 3</font>**<font 11.0pt/11;;inherit;;inherit>: la provisión de un servicio puede implicar, por ejemplo, lo siguiente:</font>

  * <font 11.0pt/11;;inherit;;inherit>Una actividad realizada sobre un producto tangible suministrado por el cliente (por ejemplo reparación de un coche)</font>
  * <font 11.0pt/11;;inherit;;inherit>Una actividad realizada sobre un producto intangible suministrado por el cliente (por ejemplo declaración de ingresos necesaria para preparar una declaración de impuestos).</font>
  * <font 11.0pt/11;;inherit;;inherit>La entrega de un producto intangible (por ejemplo, la entrega de información con el contexto de la transmisión de conocimiento)</font>
  * <font 11.0pt/11;;inherit;;inherit>La creación de un ambiente para el cliente (por ejemplo hoteles y restaurantes.</font>

**<font 11.0pt/11;;inherit;;inherit>Nota 4</font>**<font 11.0pt/11;;inherit;;inherit>: Un servicio generalmente se experimenta por el cliente</font>

==== 3.2 Conformidad : ====

Cumplimiento de un requisito.

==== 3.3 No Conformidad (NC): ====

Incumplimiento de un requisito.

==== 3.4 Requisitos: ====

Necesidad o expectativa establecida generalmente implícita u obligatoria. El término “implícita” significa que es habitual o una práctica común para la empresa, el cliente, o cualquier parte interesada.

==== 3.5 Defecto: ====

Incumplimiento de un requisito asociado a un uso previsto o especificado.

==== 3.6 Acción Correctiva: ====

Acción tomada para eliminar la causa de una no conformidad y evitar que vuelva a ocurrir.

==== 3.7 Corrección: ====

Acción tomada para eliminar una no conformidad detectada.

==== 3.8 Reproceso: ====

Acción tomada sobre un producto o servicio no conforme para hacerlo conforme con los requisitos.

==== 3.9 Reclasificación: ====

Variación de la clase de un producto o servicio no conforme, para hacerlo conforme con a requisitos diferentes de los requisitos iniciales.

==== 3.10 Reparación : ====

Acción tomada sobre un producto o servicio no conforme para convertirlo en aceptable para su utilización prevista.

==== 3.11 Desecho : ====

Acción tomada sobre un producto o servicio no conforme para impedir su uso inicialmente previsto.

==== 3.12 Concesión : ====

Autorización para utilizar o liberar un producto o servicio que no es conforme con los requisitos especificados.

==== 3.13 Permiso de Desviación : ====

Autorización para apartarse de los requisitos originalmente especificados de un producto o servicio, antes de su realización.

==== 3.14 Liberación : ====

Autorización para proseguir con la siguiente etapa de un proceso o el proceso siguiente.

==== 3.15 Servicio No Conforme (PNC) : ====

Servicio que no cumple los requisitos.

==== 3.16 Salidas No conformes (SNC) : ====

Salida que no cumple los requisitos
